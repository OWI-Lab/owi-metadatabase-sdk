# How-to guides

How-to guides are **problem-oriented** recipes. Each guide addresses a
specific task and assumes you already have a working installation.

## Installation and setup

| Guide | Problem it solves |
|-------|-------------------|
| [Install the SDK](install.md) | Install from PyPI, from source, or with optional extras |
| [Authenticate with the API](authenticate.md) | Set up token or username/password authentication |
| [Migrate from v0.10.x](migrate.md) | Update imports and dependencies from `owimetadatabase-preprocessor` |

## Working with data

| Guide | Problem it solves |
|-------|-------------------|
| [Query location data](query-locations.md) | Retrieve project sites and asset locations |
| [Query geometry data](query-geometry.md) | Retrieve model definitions, sub-assemblies, and building blocks |
| [Process a single turbine](process-single-turbine.md) | Build an `OWT` model for one turbine |
| [Process multiple turbines](process-multiple-turbines.md) | Batch-process turbines with `OWTs` |
| [Filter data with Django-style parameters](filter-data.md) | Narrow query results using keyword arguments |
