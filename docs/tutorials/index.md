# Tutorials

Tutorials are **learning-oriented** lessons that guide you through a series of steps
to complete a project. They help you get started with the OWI Metadatabase SDK
and build confidence through hands-on experience.

!!! tip "New to the SDK?"
    Start with [Your first API query](first-query.md) to learn the basics,
    then move on to [Processing turbine geometry](processing-geometry.md)
    for a deeper dive.

| Tutorial | What you will learn |
|----------|-------------------|
| [Your first API query](first-query.md) | Connect to the API, retrieve locations and geometry data |
| [Processing turbine geometry](processing-geometry.md) | Build a complete turbine model from raw geometry data |
